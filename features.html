Product Features

The S-touch is a new device that will bolster learning in classrooms through increased interaction. A list of planned features are show below:

S-Touch

Feature Progress Notes
WhiteBoard Pen
Concept stage 5%
 Will allow the S-Touch to be used just like a mobile whiteboard. The physical shell is relatively simple and a 3d printed prototype is expected by the end of next week.
Pen Software Planned* Enables users to save and load lesson plans created with the S-touch WhiteBoard Pen
Joystick Dock Planned* Will allow the S-Touch Pen to be used as a joystick when attached to this dock.
3-D Modeling Planned* Will allow the S-Touch to be used to model in three dimension with ease bu allowing the operator to use 3-Motions.
Graphic Design Planned* The S-Touch will be able to be used by graphic artists as a tablet to create detailed drawings.

*It should be noted that while some features are "Planned", there is no guarantee that the project/feature will not be replaced by a better option.
