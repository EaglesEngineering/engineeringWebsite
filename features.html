Product Features

The Origin is a new device that will bolster learning in classrooms through increased interaction. A list of planned features are show below:

Origin

Feature Progress Notes
WhiteBoard Pen
Concept stage 5%
 Will allow the Origin to be used just like a mobile whiteboard. Triangulating pen position with Arduino is being prototyped currently.
Pen Software Planned* Enables users to save and load lesson plans created with the Origin WhiteBoard Pen
Touch sensative Planned* We're looking into using pressure-sensative resistors to give the pen tip 1024 levels of pressure, similar to how Wacom tablets have touch sensitive capabilities.

*It should be noted that while some features are "Planned", there is no guarantee that the project/feature will not be replaced by a better option.
